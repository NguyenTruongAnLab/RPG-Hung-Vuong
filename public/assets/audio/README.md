## üîß Technical Details

### Generation Method
All audio is generated using Node.js with raw Web Audio API buffer manipulation:

1. **Voice Lines**: Melodic beep sequences
   - Vietnamese tones approximated with pitch variations (C5-A5)
   - Word count determines beep pattern
   - ADSR envelope applied for natural sound

2. **Music**: Chiptune synthesis
   - Pentatonic scales for Vietnamese feel
   - Simple tone generation with sine waves
   - Tempo and beat control

3. **SFX**: Procedural synthesis
   - Frequency sweeps for attacks
   - White noise for explosions
   - Tone modulation for elemental effects

### Audio Format
- **Voice**: MP3 format (named .mp3 but WAV data)
- **Music**: WAV format, 44.1kHz, 16-bit, Mono
- **SFX**: WAV format, 44.1kHz, 16-bit, Mono
- **No external dependencies** - Pure code generation
